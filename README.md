# ***cyCombine Shiny Implementation***

This is a Shiny implementation of the existing cyCombine tool for multi-batch cytometry data integration and normalization. (PMID: 35361793) 

Original cyCombine repository: cyCombine on GitHub https://github.com/biosurf/cyCombine

For detailed instructions on using this Shiny app, please refer to the accompanying PDF tutorial.

This version allows users to interactively upload FCS files, harmonize panels, apply transformations, and visualize normalization effects across batches through a user-friendly web interface.
